# 🕐 Tick Tock Time!

A fun, interactive clock-reading app for children aged 5–7.

## Features
- 🕹️ **Explore mode** — drag the hands to set the time, with real clock winding
- 🎯 **Quiz mode** — "What time is it?" and "Set the hands!" challenges
- Colour-coded hands (red = hours, teal = minutes)
- Outer minute ring showing 0–55 in 5-minute steps
- Plain English time descriptions (quarter past, half past, etc.)
